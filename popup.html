What's New

Recent updates to SF Tabs

v2.1.2
  • Storage switching fix Fixed a bug where switching between Sync and Local Storage could leave stale data behind, causing tabs to disappear if you switched storage options more than once.
  • Release notes panel New releases now show notes right here in the extension. Dismiss them once you've read them.
v2.1.1
  • Import fix Fixed a bug where importing data wasn't working as expected in v2.x.x.
  • Unique tab IDs on import Duplicate tabs in an import file now get unique IDs so deleting one doesn't remove all identical tabs.
