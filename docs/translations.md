# Translations

SF Tabs is available in multiple languages to serve our global user community.

## Available Languages

- **English** — Default
- **German** (Deutsch) — Human-reviewed translation by Andreas Bruchwitz
- **Spanish** (Español) — Human-reviewed translation by Lucas Lorenzi

## Request a Language

Would you like to see SF Tabs translated into another language? Please open an [issue on GitHub](https://github.com/chrisrouse/sftabs/issues) to request it.
