Installation

SF Tabs is available for Firefox and Chrome browsers. Choose your browser below to get started.

Firefox

  1. Visit Firefox Add-ons

    Search for "SF Tabs" in the Firefox Add-ons marketplace

  2. Click "Add to Firefox"

    Review the permissions and click the button to install

  3. Confirm Installation

    Click "Add" when prompted to confirm the installation

  4. You're ready!

    Navigate to any Salesforce Setup page to see your custom tabs

Required Permissions

SF Tabs requires the following permissions to function:

  • Access your data for sites matching: *.salesforce.com, *.force.com - Required to inject custom tabs into Salesforce Setup pages
  • Store data on your device - Required to save your custom tab configuration
  • Access browser tabs - Required to open tabs when you click on them

Chrome

  1. Visit Chrome Web Store

    Search for "SF Tabs" in the Chrome Web Store

  2. Click "Add to Chrome"

    Review the permissions and click the button to install

  3. Confirm Installation

    Click "Add extension" when prompted

  4. You're ready!

    Navigate to any Salesforce Setup page to see your custom tabs

Required Permissions

SF Tabs requires the same permissions as the Firefox version (see above).

Verifying Installation

To verify SF Tabs is installed correctly:

  1. Navigate to any Salesforce org
  2. Click the Setup gear icon (⚙️) and select "Setup"
  3. Look for the SF Tabs icon in your browser toolbar
  4. You should see default tabs in the Salesforce Setup menu
