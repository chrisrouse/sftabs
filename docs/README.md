# SF Tabs Documentation

This directory contains the documentation for SF Tabs, built using Jekyll with the Just the Docs theme.

## Local Development

To run the documentation site locally:

1. Install dependencies:
   ```bash
   cd docs
   bundle install
   ```

2. Run Jekyll:
   ```bash
   bundle exec jekyll serve
   ```

3. Open your browser to `http://localhost:4000/sftabs/`

## GitHub Pages

This site is automatically built and deployed by GitHub Pages when changes are pushed to the `main` branch.

## Structure

- `_config.yml` - Jekyll configuration
- `index.md` - Home page
- `installation.md` - Installation guide
- `user-guide.md` - User guide
- `shortcuts.md` - Keyboard shortcuts guide
- `configuration.md` - Configuration reference
- `Gemfile` - Ruby dependencies

## Theme

This site uses the [Just the Docs](https://just-the-docs.com/) Jekyll theme.
